## 💬 AI Cevap Örnekleri

### Doğru Cevap ✅
```json
{
  "texts": {
    "Product design": "Ürün Tasarımı",
    "Contact": "İletişim"
  }
}
```

### Yanlış Cevap ❌ (Markdown ile)
```json
{
  "texts": {
    "Product design": "Ürün Tasarımı"
  }
}
```
**Çözüm:** Sadece `{` ile `}` arasını kopyala

### Yanlış Cevap ❌ (Açıklama ile)
```
İşte çevrilmiş JSON:
{
  "texts": {...}
}
Bu çeviri profesyonel...
```
**Çözüm:** Sadece JSON kısmını al
